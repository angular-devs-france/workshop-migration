## Manuel solution

### Prerequisites

- [Node.js](https://nodejs.org/en/download/)
- [Angular CLI](https://angular.io/cli)

### Install dependencies

```bash
npm install
```

### Start the server

```bash
npm run server
```

### Start the Angular application

```bash
npm run start
```
